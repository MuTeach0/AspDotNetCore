# M01.BasicSetup

This project demonstrates the basic setup of an ASP.NET Core Web API using controllers. It serves as an introduction to API development with ASP.NET Core.

## Project Features

- Basic Web API project structure
- Simple ProductController implementation
- Fundamental HTTP method usage (GET)
- Basic routing setup
- Development environment configuration

## Key Components

### ProductController
- Basic controller implementation
- Simple GET endpoint
- Route configuration
- HTTP method attributes

### Program.cs
- Minimal API setup
- Service configuration
- Middleware setup

## How to Run

1. Open the solution in Visual Studio
2. Set M01.BasicSetup as the startup project
3. Press F5 or click Run to start the application
4. Use the provided request.http file to test the API

## API Endpoints

- GET /api/product - Returns a simple product list

## Learning Outcomes

- Understanding basic Web API project structure
- Learning controller-based routing
- HTTP method implementation
- Basic API testing using request.http files
