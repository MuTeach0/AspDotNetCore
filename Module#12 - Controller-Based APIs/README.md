# Module 12 - Controller-Based APIs

This module covers the implementation of REST APIs using ASP.NET Core Controllers. The module is divided into three main projects that progressively build up knowledge of API development.

## Projects Overview

### M01.BasicSetup
Basic setup of an ASP.NET Core Web API project with a simple ProductController demonstrating fundamental controller concepts.

### M02.BuildingRESTFullApi
A complete RESTful API implementation with proper data models, repositories, and CRUD operations. Includes pagination and related entities (Product Reviews).

### M03.ContentNegotiation
Advanced API features focusing on content negotiation, custom formatters, and different output formats including a custom plain text table formatter.

## Key Learning Points
1. Controller-based API development in ASP.NET Core
2. RESTful API design principles
3. Data modeling and repository pattern
4. Request/Response models and DTOs
5. Content negotiation and custom formatters
6. API documentation and testing using HTTP request files

## Technologies Used
- ASP.NET Core
- C#
- HTTP REST APIs
- CSV Data Storage
- Content Negotiation
