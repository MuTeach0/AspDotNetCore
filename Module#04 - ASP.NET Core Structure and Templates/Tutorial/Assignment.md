Module Assignment — ASP.NET Core Configuration (Module 04)

تمارين شاملة:

1) قارن بين ترتيب أسبقية مصادر التهيئة (appsettings.json, environment variables, command line, in-memory, user-secrets). وضّح أي مصدر يأخذ الأولوية ولماذا.

2) اختَر مشروعًا واحدًا (مثلاً `FileProviders`) وغيّر طريقة القراءة بحيث تفضّل متغير بيئة على قيمة من الملف. فسّر التغييرات التي أجريتها.

3) نفّذ تجربة تجمع: اضبط secret باستخدام `dotnet user-secrets` في `UserSecretProvider`، واضبط أيضًا قيمة بنفس المفتاح في `appsettings.json`، ثم بيّن أي قيمة تُقرأ عند التشغيل ولماذا.

4) أضف ملف `README.md` و`Assignment.md` لأي مشروع جديد تضيفه وتأكد أن التعليمات واضحة لتشغيله.

تحقق: لكل تمرين سجّل خطواتك، الأوامر التي نفّذتها، والاستجابات المتوقعة.
