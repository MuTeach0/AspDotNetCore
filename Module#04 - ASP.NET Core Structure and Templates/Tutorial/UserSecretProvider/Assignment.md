Assignment — UserSecretProvider

1) شغّل التطبيق واطلب `/ServiceName`. ماذا يظهر؟

2) إذا أردت تخزين قيمة سرية محليًا بدلاً من `appsettings.json`، ما الخطوات لاستخدام `dotnet user-secrets`؟ اكتب الأوامر المطلوبة (PowerShell).

3) جرّب إضافة secret محليًا (مثال: `dotnet user-secrets set "ApiKey" "secret123"`) ثم اقرأه عبر `IConfiguration`.

4) دوّن ملاحظة قصيرة لماذا يجب استخدام User Secrets في التطوير وليس في الإنتاج.
