Assignment — CommandLineProvider

1) شغّل المشروع وجرب استدعاء `/ServiceName`. ماذا تسترجع؟

2) أعد تشغيل التطبيق مع وسيطة سطر أوامر لتعريف قيمة جديدة: مثال (من مجلد المشروع):
```
dotnet run --ServiceName="CLI Service"
```
ثم اطلب `/ServiceName` وتحقق من النتيجة.

3) جرّب إضافة مفتاح متعدد المستويات (مثل `Logging:LogLevel:Default`) عبر سطر الأوامر وانظر كيف يظهر عند طلب المفتاح.

4) ضع ملاحظة قصيرة توضح متى يكون مفيدًا استخدام مزود سطر الأوامر مقابل appsettings أو متغيرات البيئة.
