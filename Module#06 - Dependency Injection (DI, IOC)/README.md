# Module 06 — Dependency Injection (DI, IOC)

هذا الموديول يحتوي على أمثلة ومشروعات قصيرة لشرح مبادئ الـ Dependency Injection، Inversion of Control، وService Lifetimes في .NET.

المشروعات داخل المجلدات `M01.DependencyHell` ... `M12.ServiceLifeTime`.

كيفية التشغيل (عام):

- لتشغيل أي مشروع من داخل الموديول استخدم:

```powershell
dotnet run --project ./M01.DependencyHell/M01.DependencyHell.csproj
```

(استبدل اسم المشروع بالمجلد المراد).

ماذا يوجد هنا؟
- شروحات قصيرة ومشروعات تجريبية لكل مفهوم.
- ملفات `Program.cs` و `.csproj` في كل مشروع.

اقرأ ملفات `README.md` الخاصة بكل مشروع لمزيد من التفاصيل.
