# Assgnment — M02.CreateMiddlewareUse

أسئلة ومهام:

1) اشرح الفرق بين `app.Use(...)` و `app.Run(...)` من حيث التحكم في مرور الطلب.
2) مهمّة عملية: أضف Middleware عبر `Use` يضيف رأسًا (header) جديدًا في الاستجابة، ثم اختبر وجوده.
3) صِف كيف ولماذا ستستخدم `next()` داخل `Use` middleware.

مَخرَجات متوقعة:
- لقطة شاشة من استجابة HTTP تحتوي على الهيدر الجديد.
