# Module Assignment — Middlewares (Module#07)

أسئلة ومهام لتأكيد فهمك للموديول ككل:

1) اشرح بخطوط عامة كيف يعمل خط الـ Middleware في ASP.NET Core (كيف يُمرَّر الطلب، ما وظيفتا `next()` و `context`).
2) قارن بين `Use`, `Run`, `Map`, `MapWhen`, و `UseWhen` — ضع مثالًا عمليًا صغيرًا لكُلٍ منها.
3) مهمّة عملية شاملة: اختر مشروعين من الموديول (مثلاً M05 و M06)، وادمجهما في مشروع واحد بحيث ترى تأثير الترتيب والـ chaining مع فروع مختلفة — دوّن ملاحظاتك.
4) اذكر ثلاث أخطاء شائعة عند التعامل مع middlewares وكيف تتجنّبها.

اقتراح تسليم:
- ملف نصي أو PDF يضم الإجابات، ولقطات شاشة من الـ console ونتائج طلبات HTTP تثبت تنفيذ المهام العملية.
