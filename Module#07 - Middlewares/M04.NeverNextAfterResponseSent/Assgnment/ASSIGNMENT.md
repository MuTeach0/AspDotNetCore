# Assgnment — M04.NeverNextAfterResponseSent

تمارين وأسئلة:

1) اشرح لماذا قد يكون استدعاء `next()` بعد إرسال استجابة مشكلة (أعطِ مثالاً).
2) مهمّة: اكتب middleware يقوم بإرسال استجابة ثم يحاول استدعاء `next()` — لاحظ السلوك واكتب ملاحظاتك.
3) كيف يمكن تفادي المشاكل المتعلقة بكتابة الاستجابة مرتين؟ اذكر 2 استراتيجيات.
