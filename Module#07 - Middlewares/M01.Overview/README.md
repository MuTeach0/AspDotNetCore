# M01.Overview

ملف توضيحي لمشروع M01.Overview

ماذا يفعل المشروع
- يقدم مثال بسيط يشرح مفهوم الـ Middleware في ASP.NET Core.

كيفية التشغيل
1. افتح الطرفية في مجلد المشروع:
   - `cd "M01.Overview"`
2. شغّل التطبيق:
   - `dotnet run`

الملفات المهمة
- `Program.cs` - نقطة الدخول وبناء خط الـ Middleware.
- `request.http` / `reqest.http` - أمثلة لعمل طلبات HTTP لاختبار التطبيق.

ما الذي يجب ملاحظته
- أين يتم تسجيل الطلبات (logs)؟
- كيف يمر الطلب عبر الـ middlewares؟
- هل يتم إرسال استجابة من middleware أولي أم يمر لباقي السلسلة؟
